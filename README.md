# Air-Tunnel
This is a wind tunnel. It is pull-based so that a more laminar-flow can occur. It has a grate-looking thing at the front to reduce varying wind turbulence. It will have a mister at the fromt to provide a visual flow of air. The fan drags air through at high speeds. I reduced the size of the center of the enclosure to incease the air flow speed.
